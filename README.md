Startup Funding Success Probability

AI & Intership Project

Problem Statement

Thousands of startups launchs every year,but only a small percentage secure funding successfully.Investors and startups accelerators need a data driven approach to idential startups with high growth potential and lower investment risk.
This project analyzes starups datasets using statistics,probability analysis, and data visualization techniques to identify factors influencing startups funding success.

Objective

The main objectives of this project are:

1.Analyze startup funding data
2.Identify factors affecting funding success
3.calculate funding probabilities
4.perfrom statistical analysis
5.Visualize relationships between variables
6.Generate business insights and recommendations
